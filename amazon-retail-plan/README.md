# Amazon-Retail-Projekt

Eigenständiges Projekt, unabhängig von der TagCore-Portfolio-Website in diesem Repository. Bitte **nicht** in `netlify.toml`, `site.config.yaml` oder `scripts/` einbinden — dieser Ordner ist bewusst isoliert vom Website-Build.

## Geschäftsmodell

Online-Arbitrage/Wholesale-Sourcing + Amazon FBA: Ware wird extern in großer Stückzahl günstig eingekauft, ins Amazon-FBA-Lager geschickt (keine eigene Lagerung) und dort mit Marge verkauft. Details in [`phase-1-niche-strategy/niche-strategy.md`](phase-1-niche-strategy/niche-strategy.md).
